# My Proyect

Sitio web estático para barbería (o negocio similar) desarrollado con HTML, CSS y JavaScript. Incluye integración de Bootstrap

## Estructura del proyecto

- `index.html` — Página principal
- `style.css` — Estilos personalizados
- `script.js` — Lógica JavaScript
- `imagenes/` — Imágenes usadas en el sitio

## Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, etc.)
- (Opcional) Git para control de versiones

## Personalización

- Modifica `style.css` para cambiar estilos propios.
- Edita `index.html` para actualizar el contenido.
- Agrega imágenes en la carpeta `imagenes/`.
